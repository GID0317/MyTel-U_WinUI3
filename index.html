This application was developed in response to the absence of a native desktop application for MyTel-U. Utilizing Microsoft's WinUI3 fluent design, now you can use My Tel-U natively on the desktop!

How Can I Download It?

Go to the Release Page > Click at the Assets > Download The Latest Version (Recommended) or Download Appropriate Versions You Need

Third-party Packages Used

  • CommunityToolkit
  • Win App SDK
  • WinUIEX
  • DotNet

NOTE

This project is entirely community-driven and is not affiliated with MyTel-U in any way. This project is open-sourced so any contributions are welcomed!

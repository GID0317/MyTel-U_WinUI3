Aplikasi ini dibuat oleh komunitas MyTel‑U Launcher untuk dijalankan di desktop modern Windows 10 atau 11. Dikembangkan menggunakan desain Fluent WinUI 3 dari Microsoft, kini Anda dapat menggunakan My Tel‑U di desktop Anda!

Proyek ini bersifat open source, jadi kontribusi sangat disambut! （￣︶￣）↗

> Atau [Unduh Versi yang Sesuai dengan Kebutuhan Anda](https://github.com/GID0317/MyTel-U_WinUI3/releases "Download Apropriate Versions You Need")

# Fitur
- **Beranda**: Pusat Aplikasi, di-porting dari MyTel‑U untuk perangkat mobile.  
- **Kehadiran**: Cek kehadiran secara berkala selama semester.  
- **Jadwal**: Akses jadwal mata kuliah semester berjalan.  
- **Alat Komunitas**: Jelajahi alat yang terdaftar di situs web laboratorium dan situs alat buatannya komunitas.

## Preparation Before Download

Before downloading MyTel‑U, make sure your system meets these requirements.  
You can verify your device specs by opening **Settings > System > About**.

## Download Process

To download the My Tel‑U Launcher:

1. Navigate to the [Releases](https://github.com/GID0317/MyTel-U_WinUI3/releases) page.
2. Download the file named `MyTel_UProto_x.x.x.x_x64.zip`.
3. Once downloaded, extract the ZIP file by right-clicking on it, selecting **Extract All**, and clicking **Extract**.

# Installation

Follow these steps to install the My Tel‑U Launcher on your Windows system.

## Table of Contents
1. [Installing the Certificate](#installing-the-certificate)
2. [Installing the Application](#installing-the-application)
3. [Running the Application](#running-the-application)

## Installing the Certificate

After extracting the download, locate the certificate file: `MyTel_UProto_x.x.x.x_x64.cer`.

1. Double-click the `.cer` file.
2. Click **Install**.
3. Select **Local Machine** and click **Next**.
4. When the UAC dialog appears, click **Yes**.
5. Click **Browse**, choose **Trusted Root Certification Authorities**, then press **OK**.
6. Click **Next** and then **Finish**.

## Installing the Application

After the certificate installation, continue with the following steps:

1. Double-click the `MyTel_UProto_x.x.x.x_x64.msix` file.
2. Follow the on-screen instructions to complete the installation.

Once installed, you can find the My Tel‑U Launcher in your Start Menu—typically under **Recommended** or **All Apps**.
